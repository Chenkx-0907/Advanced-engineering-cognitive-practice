### 夏季课程大作业

2023.6.15之前提交一份2000字的调研报告，主题包括：

* 中国与其他国家的自动驾驶技术路线，各自的特点，哪些地方不同，为什么不同，以及具体做法。
* 国家国情政策，不同企业选取的技术路线等等。
* 其他思想。
